Controls register (summary)

# Control objective Control Evidence artefact Review cadence
C-01 CO₂e / 1k requests ≤ budget Carbon Budget Gate (CI) carbon-budget.yml run summary Every PR; monthly trend
C-02 Budget inputs traceable and versioned Workbook CSV in version control; PR review required appendix-d-baseline-improved.csv On change; quarterly audit
C-03 Grid intensity reflects real-world UK data Daily automated refresh via GitHub Actions grid_intensity_uk_summary.json Daily (automated); monthly spot-check
C-04 Endpoint behaviour is evidenced Versioned probe run summary committed to repo probe_run_summary.json Per evidence refresh (≤ weekly)
C-05 Secrets never committed .gitignore excludes .env; SECURITY.md policy; CodeQL scans .gitignore, SECURITY.md Continuous; quarterly review
C-06 Boundary and assumptions documented Appendix C under version control appendix-c-boundary.md On assumption change; quarterly
C-07 Data sources documented and reproducible Appendix D with fetch script committed appendix-d-data-sources.md On source change; quarterly
C-08 Risks identified and mitigated Risk register maintained appendix-g-risk-register.md Quarterly risk review
C-09 KPIs defined and reportable KPI definitions and cadence documented appendix-f-kpis.md Quarterly or on KPI change
C-10 Stakeholder accountability defined RACI table maintained appendix-e-raci.md Annually or on team change
C-11 Architecture decisions recorded ADR set in docs/adr/ ADR-001 · ADR-002 · ADR-003 On architectural change
C-12 Operator procedures documented Runbook: validate gate, rotate keys, refresh evidence, pause spend runbook.md Quarterly; on procedure change

Architecture decision records

  • ADR-001 — Carbon Budget Gate in CI
  • ADR-002 — Evidence pack and static monitoring dashboard
  • ADR-003 — Optional live endpoint and secret handling

Review cadence

  • Every PR: Carbon Budget Gate (C-01) — automated.
  • Daily: Grid intensity refresh (C-03) — automated via GitHub Actions.
  • Monthly: gCO₂e/1k trend, total estimated CO₂e (C-01, C-02, C-04).
  • Quarterly: Full governance review — budgets, exceptions, risks, ADR updates (C-02, C-05–C-12).
  • On incident: Key rotation and secret audit (C-05).
